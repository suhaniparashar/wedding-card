Quick Start
----------
1. To view the invitation locally:
   - Using Python's built-in server:
     ```
     cd wedding
     python -m http.server 8000
     ```
   - Then open http://localhost:8000/weeding.html in your browser

2. To view directly:
   - Simply open weeding.html in any modern web browser

Features
--------
1. Elegant Design
   - Ivory and gold color scheme with maroon accents
   - Beautiful typography using Great Vibes for names
   - Subtle animations and transitions
   - Decorative corners and background patterns

   - Header with couple's names
